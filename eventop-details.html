Project Overview

Eventop is a comprehensive event booking and management system designed to streamline the process of organizing and managing events. This project was developed as part of my academic coursework, focusing on creating a robust web application that handles event registration, venue management, and user interactions.

The system provides a user-friendly interface for both event organizers and attendees, featuring real-time booking capabilities, payment integration, and comprehensive event management tools. Built with modern web technologies, Eventop demonstrates my ability to create scalable, user-centric applications.

Key Features

Event Management

Comprehensive event creation, editing, and management system with detailed event information and scheduling capabilities

Payment Integration

Secure payment processing with multiple payment gateways including VNPay, ZaloPay, and PayPal for seamless transactions

Real-time Chat

Integrated chat functionality using SignalR for real-time communication between event organizers and attendees

User Management

Comprehensive user registration, authentication, and profile management system with role-based access control

Venue Management

Advanced venue booking and management system with availability tracking and location-based services

Analytics Dashboard

Comprehensive analytics and reporting tools for event performance tracking and business insights
